Lilyhammer Rendering Engine

Matthias Moulin

Description

A ray tracing engine written from scratch, capable of rendering models (specified in .obj format) and scenes (specified in .xml format) with several kinds of effects (such as reflection and refraction) by using a variety of acceleration data structures (such as BVHs, kd-trees, regular grid). All code is written in Java.
